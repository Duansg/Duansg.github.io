【DIY无人机】电调固件升级[已收录]

电调


为什么这里要说电调，因为电调对于多旋翼来说肯定是响应越快越好,多轴要的就是快速响应!但快速响应等高性能就意味着价格高,所以我们在价格和性能之间往往是不好选择的(就是因为穷,哈哈哈哈),但 BLHELI 电调开源固件解决了这个问题。我们只需要花 40-50 块钱买一个好盈天行者 20A/40A 电调，然后往电调里刷 BLHELI 的电调固件,然后你就得到了一个响应速度可以和 200-300 块钱的高价电调相当的廉价电调!话说回来这 BLHELI 固件也不是非刷不可,不刷就是响应慢了点,照样能飞起来!好盈天行者 20A 和好盈天行者 40A,选哪个都行,看你资金情况和长远考虑,其实好盈天行者 20A 就目前来说就够用了! 电调原固件可以,但响应不快!刷了 BLHeli 固件之后响应速度大大提高,这会让你四轴飞行更加稳定!
固件下载：http://www.helifreak.com/blog.php?b=2001
BLHeli-Setup 下载：https://app.box.com/s/z32i4rehild8vsbcnr5x

下载器

要往电调里刷固件,需要先做一个下载器,文档里有相应的引脚连接方法，我们选择 Arduino Nano 来作为中转下载器。Arduino Nano 从某宝上购买即可,几元钱,我买的还是改进版,大家买的时候注意一下即可。这个连接到电脑上打开 IDE 会自动下载驱动,如果没有驱动,可以给店家索取,他们都会提供的。

然后打开 BLHeli-Setup

然后选择 nano,原版的 Baud 请选择 57600,改进版的是 115200


成功之后如下

电调连接

这里需要把电调的热缩管拆开,但是你可以开天窗,就是用剪子剪掉一个正方形的洞,但是这样你不好找位置,还容易把贴片元器件损坏,所以就直接全部剪掉热缩管就可以了,刷完最好在用热缩管封起来。

然后找到这个触点

然后找到 Nano 的引脚示意图


然后给电调供电，把上图的连接线连接起来,笔者使用 B6 平衡充进行供电,实际可以使用电池,但是需要注意是,因为触点不要连接,动手能力差一点的可以找两个人配合，笔者因为这里需要一只手连接触点，一只手刷固件，就没有拍照。

固件烧录

选择 Setup–>Read Setup From Hex File

选择电调

选择 Skywalker 20A_MULTI

如图设置

选择刷新

再次选择 Skywalker 20A_MULTI,然后一直点确定就可以了。

成功之后如下


刷完之后，连接上电机，通电之后会有一段自检的音乐，反正跟之前的是不一样的,然后可以试下电机的响应速度之类的，你就会发现比之前提升了不少。
