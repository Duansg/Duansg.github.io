Permission denied (publickey)解决办法

刚新建了个笔记项目，准备上传却遇到这个问题，更新 key 也不行，找了好久终于找到解决办法。

远程仓库的链接 SSH

ssh 准备连接远程服务器却遭提示” Permission denied (publickey) “, 这是由于您没有将公钥( publickey ) 添加到本地 ssh 环境造成的，或者是由于多日未 进行 ssh 登录操作，本地 publickey 失效造成的。只要 使用 ssh-add 命令再次添加一下公钥即可。
ssh-add /Users/duansg/.ssh/id_rsa
